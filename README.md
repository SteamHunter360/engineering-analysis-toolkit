# Engineering Analysis Toolkit

A Python-based engineering toolkit containing mechanical engineering calculations and visualisations.

The purpose of this project is to build practical engineering software for structural, mechanical, thermal and fluid engineering applications.

## Current Modules

- Beam Deflection Analysis
- Shaft Stress Analysis
- Mohr's Circle
- Buckling Analysis

## Technologies

- Python
- NumPy
- Matplotlib

## Future Modules

- Torsion Analysis
- Pressure Vessel Analysis
- Fatigue Analysis
- Heat Transfer
- Fluid Flow
